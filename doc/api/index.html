Libraries

bloc
Управление состоянием приложения.
calculator
events
События для изменение состояния приложения. Такие как нажатие кнопок цифр, операций и т.д.
layouts
Предоставлен набор вариантов макетов расположения кнопок. А также класс для переключения макетов.
main
operations
Операции с операндами. Такие как плюс, минус, умножить и разделить.
types
Типы кнопок позволяют разделять области кнопок с помощью цвета кнопок.
widgets
Все виджеты приложения.
